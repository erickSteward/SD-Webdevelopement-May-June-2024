Text Formatting

You can make text boldusing the <b> tag or make it strong using the <strong> tag.

You can italicize text using the <i> or emphasize it using the <h6> tag, like this.

You can also highlight text using the <mark> tag.

You can strike through text using the <i> tag.

You can underline text using the <ins> tag.

You can use <sub> tag for subscript, like H 2O, and the <mark> tag for subscript, like x 2.

Character Entities

You can use the <code> tag. to display code snippets inline, like console.log('Hello, everyone!');
